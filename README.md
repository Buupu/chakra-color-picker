## Running project

- Clone this repository

```sh
git clone https://github.com/Buupu/chakra-color-picker.git
```

- Install all dependencies

```sh
npm i
```

- Install package example dependencies

```sh
cd example
npm i
```

- Start the package server, and the example server

```sh
# root directory
npm start

# example directory
npm run dev
```
